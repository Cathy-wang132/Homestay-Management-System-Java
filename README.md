#### 3、技术栈与工具

本项目采用主流技术栈与现代化开发工具，保证开发效率与系统稳定性：

·  后端： Java 21 + Spring Boot 2.7 框架

·  数据库： MySQL 8.0

·  构建工具： Maven 3.9

·  前端： Vue 2 + Element UI

·  开发工具： VSCode 与 IntelliJ IDEA

·  环境要求：

o  Node.js ≥ 16.20

o  Java ≥ 21

o  Spring Boot 2.7

o  MySQL 8.0

o  Maven 3.9
